Placez vos pistes MP3 ici.
Noms par défaut attendus:
- chill.mp3 (musique douce / calme)
- stress.mp3 (musique tendue / stress)

Option avancée: vous pouvez configurer d’autres chemins en définissant window.MUSIC_TRACKS dans index.html avant le script /src/main.js.
Exemple:

<script>
  window.MUSIC_TRACKS = {
    chill: 'assets/music/ambient_lofi.mp3',
    stress: 'assets/music/techno_drive.mp3',
  };
</script>
